# HTML as a Programming Language
## Syntax
```html 
<!DOCTYPE haapl>
<haapl>
    <head>
        <version>0.0.1</version> <!-- HAAPL Version -->
    </head>
    <body>
        <func name="main">
            <def name="message">"hello, world!"</def>
            <call name="print">
                <value>message</value>
            </call>
        </func>
    </body>
</haapl>
```
